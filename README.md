# Shooter_Game
Shooter game is a game developed from python.

How to play:
prees on the right and left arrow to move the ship.
prees on the space button to blast bullets from the ship and kill the monsters.

Rules of the game:
if a monster touches the ship you lose.
if 5 monsters passed under the ship you lose.
if you kill 15 monsters using the bullet you lose.

About the game:
the game is about 2 characters the ship and the monster.
you have to avoid touching monsters.
there are 2 counters at the top left of the game it says the score.
the background is in space(vacume)
if you win or lose the game will restart in less than a secound.
there are sounds in the game.
